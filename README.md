# React Geo Weather

> Built during ReactJS Boston Meetup on Wednesday, October 17 2018.

Uses `navigator.geolocation` and worldweatheronline APIs with the React 16
`<ErrorBoundary>` component.

## Install

1. Clone repo
1. `yarn`
1. `yarn start`
